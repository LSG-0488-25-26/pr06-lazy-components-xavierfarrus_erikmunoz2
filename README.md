Este proyecto consiste en una colección de cartas de Yu-gi-oh, en la lista podemos encontrar algunas de las cientas de cartas que existen en el juego de Yu-gi-oh.


<img width="406" height="899" alt="image" src="https://github.com/user-attachments/assets/f34b7004-8860-475a-a04e-37101bbed126" />


Una vez elegimos cualquiera de ellas, entramos en un detalle donde nos aparece el nombre de la carta, el tipo de carta que es (carta trampa, carta de hechizo y monstruo), el ataque y defensa, si es un monstruo aparecen valores numéricos y si es un hechizo salen guiones ya que no dispone de esos valores y para acabar una descripción de lo que hace la carta. También sale una imagen ampliada de la carta para poder verla mas de cerca.
